Linux下文件的压缩

#####压缩文件概述

  1. gzip （针对单一文件）

    基本概念：当你使用gzip进行压缩时，源文件就不存在了，在linux下压缩的gz文件，在windows系统下可以使用WinRAR解开
    命令用法：

    • 压缩 gzip file压缩file文件，输出file.gz
    • 解压 gzip -d file.gz解压file.gz文件，输出file

  2. bzip2 （针对单一文件）

    基本概念：bzip2是为了替换gzip，提供更佳的压缩比而来的，其基本用法和gzip一致
    命令用法：

    • 压缩 bzip2 file压缩file文件，输出file.bz2
    • 解压 bzip2 -d file.bz2解压file.bz2文件，输出file

  3. tar （打包多个文件，通过添加额外参数，同时进行压缩）
    基本概念：tar是一种打包命令，能够将多个文件或目录打包成一个大文件的命令功能，使用tar进行打包压缩的文件，以gzip压缩为例，生成*.tar.gz文件，这种文件在windows系统下能够用WinRAR解开
    命令用法：（以gzip压缩为例 参数 -z）

    • 压缩 tar zcvf file.tar.gz file1 file2 file3 将file1、file2、file3通过tar打包，并使用gzip压缩，这里面-z表示gzip程序压缩，-c表示压缩，-v表示显示正在处理的文件名，-f后接自己取的压缩文件名 ，注意！这里的gzip压缩会保留源文件
    • 解压 tar zxvf file.tar.gz -C /tmp/test/ 将文件解压到指定目录-x表示解压，-C表示指定目录
