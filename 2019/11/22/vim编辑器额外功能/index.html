vim编辑器额外功能

2019-11-22
Linux

可视化区块（Visual Block）

以一个矩形为操作单位，进行复制、删除、粘贴操作

问题描述


想要将上图处理成下图的形式

解决方案

a. 按住ctrl+v，此时会以矩形的方式将光标进过的地方变白；
b. 将变白的地方复制y；

c. 移动到第一行末尾，插入一个空格，再回到命令行模式，粘贴p。

多窗口功能

可以通过将一个窗口划分为多个，然后同时查看和编辑多个文件的功能

命令操作

:sp filename2 在已打开的文件filenam1输入，可以将编辑窗口划分为2个，同时编辑两个文件filename1和filename2
ctrl + w + ↑ 移动到上方窗口
ctrl + w + ↓ 移动到下方窗口
ctrl + w + q 结束退出当前窗口

关键词补全功能

其实感觉vim的关键词补全有点麻烦，先记下来，可能以后用得上

补齐命令组合

ctrl + x -> ctrl + n 通过正在编辑的这个文件的内容文字作为关键词，予以补齐
ctrl + x -> ctrl + o 需要提前制定文件扩展名，然后以扩展名作为语法补充，比如html扩展名，则自动补齐html的相关关键词
